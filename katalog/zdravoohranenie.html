Вопросы и ответы

Как стать врачом УЗИ?

Специальность «Ультразвуковая диагностика» можно получить двумя способами: либо пройти курс профессиональной переподготовки (если Вы соответствуете квалификационным требованиям приказа №206н Минздрава России), либо окончить ординатуру по УЗИ.

Сколько длится переподготовка по УЗИ?

Переподготовка по ультразвуковой диагностике длится 3 месяца, из которых 2,5 месяца длится лекционный блок и 2 недели длится интенсивная практика с постановкой руки и отработкой мануальных навыков.

В каком городе можно пройти переподготовку по УЗИ?

Очные курсы переподготовки по УЗИ можно пройти в Москве, Санкт-Петербурге, Ростове-на-Дону, Краснодаре, Сочи, Махачкале. Также открываются курс и в других городах России – уточняйте у наших методистов.

Кто ведет на курсе переподготовки по УЗИ?

В реализации курсов переподготовки по ультразвуковой диагностике принимают участие все сотрудники кафедры УЗИ АПКиПП.

На каких УЗИ аппаратах происходит обучение врачей УЗИ?

Переподготовка по УЗИ осуществляется на УЗИ аппаратах разных брендов и разных классов: Миндрей DC-45, Миндрей М5, Самсунг V8.

Какой документ выдается после обучения по УЗИ?

После переподготовки по УЗИ Вы получите диплом о профессиональной переподготовке по ультразвуковой диагностике с правом ведения деятельности и с присвоением квалификации «врач ультразвуковой диагност».

Сколько часов должно быть в дипломе по УЗИ?

Не менее 576 академических часов. В Академии реализуется программа в объеме 600 академических часов.

По каким темам есть очные стажировки для врачей УЗИ?

Очные стажировки на кафедре УЗИ проходят по широкому кругу тем: эластография, стеатометрия, эхокардиография, УЗИ сосудов, УЗИ в гинекологии, УЗИ в урологии, УЗИ поверхностно расположенных органов, применением контрастных веществ в ультразвуковой диагностике и другое.

Сколько длится очная стажировка по УЗИ?

Очные стажировки на кафедре УЗИ, как правило, длятся 4 дня.

Есть ли курс по УЗИ в гинекологии?

Да, на кафедре реализуется очная стажировка на тему «УЗД женского здоровья».

Есть ли курс по УЗИ в урологии?

Да, на кафедре реализуется очная стажировка на тему «УЗД мужского здоровья».

Есть ли курс по эхокардиографий для начинающих?

Да, на кафедре реализуется очная стажировка на тему «Эхокардиография для начинающих».

Есть ли у Вас программа ординатуры по УЗИ?

Да, программа ординатуры по УЗИ реализуется на базе АНО ВО «Институт МедСкиллс».

В какой форме проходит обучение в ординатуре по УЗИ?

Обучение в ординатуре по УЗИ проходит в очной форме.

На какой клинической базе происходит обучение по УЗИ?

Обучение по УЗИ проходит на нескольких клинических базах – в медицинских организациях разных типов и видов – больницах, поликлиниках, частных клиниках – по договорам на клиническую базу практики, включая собственную клиническую базу Академии – клинику «МедСкиллс».
